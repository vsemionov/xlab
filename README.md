# XLab
Transformer Lab

## Usage
### Train
```shell
./xlab.py fit -c conf/xlab.yaml
```

### Validate
```shell
./xlab.py validate -c conf/xlab.yaml --ckpt_path logs/version_0/checkpoints/last.ckpt
```
